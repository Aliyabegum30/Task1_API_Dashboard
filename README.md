# 🌤 Weather Dashboard

A modern weather dashboard that shows real-time weather information for any city in the world using the OpenWeather API.

## Features

* 🔍 Search weather by city
* 📍 Auto detect location using GPS
* 🌅 Sunrise and sunset time display
* 🌞 UV Index information
* 🌫 Air Quality Index
* ⏳ Hourly weather forecast
* 📅 5-day weather forecast
* 🌧 Animated rain effect
* 🎨 Dynamic background based on weather
* 💡 City suggestions while typing
* ⚡ Quick city buttons

## Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeather API

## How it Works

1. Enter a city name in the search bar.
2. Press **Enter** or click the search icon.
3. The dashboard fetches real-time weather data from the OpenWeather API.
4. Additional data such as UV index and air quality are fetched using latitude and longitude from the weather response.

## APIs Used

* Current Weather API
* 5 Day Forecast API
* UV Index API
* Air Pollution API

All APIs are provided by OpenWeather.
